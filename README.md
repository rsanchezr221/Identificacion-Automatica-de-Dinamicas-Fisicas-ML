# Identificación Automática de Dinámicas Físicas mediante Machine Learning

## Descripción del proyecto

Este proyecto desarrolla un pipeline de aprendizaje automático para identificar automáticamente fenómenos físicos a partir de trayectorias posición-tiempo.

La idea central consiste en simular mediciones experimentales de un sistema dinámico, incorporar ruido asociado a la adquisición de datos y utilizar técnicas de Machine Learning para clasificar el tipo de movimiento responsable de cada trayectoria.

El proyecto busca reproducir un escenario similar al análisis de datos experimentales, donde un sistema de medición proporciona únicamente la evolución temporal de una variable física y se desea inferir la dinámica subyacente.

Los fenómenos considerados son:

- Movimiento uniforme.
- Movimiento uniformemente acelerado.
- Movimiento oscilatorio.

---

# Objetivo

Desarrollar un modelo capaz de identificar automáticamente la dinámica física de una trayectoria utilizando características derivadas de la cinemática.
